# Grocery Meal Planner

A Python-based meal planning and grocery optimization project that starts with nutrition data and evolves into a complete meal planning application.

## Project Goal

Build a system that can:

- Store and analyze nutrition data
- Categorize foods for meal planning
- Generate meals based on nutrition targets
- Create grocery lists
- Incorporate food pricing from APIs
- Optimize meal plans by cost and nutrition
- Provide dashboards in Power BI
- Eventually become a web application
